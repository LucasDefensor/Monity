# Monity

Aplicação fullstack para controle de gastos pessoais, desenvolvida com foco em simplicidade, organização financeira e boas práticas de desenvolvimento.

O projeto permite registrar, visualizar e gerenciar despesas, com integração entre frontend e backend via API REST.

---

## 🚀 Tecnologias utilizadas

- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Comunicação:** Axios  
- **Versionamento:** Git e GitHub  

---

## ⚙️ Funcionalidades

- Cadastro de gastos  
- Listagem de despesas  
- Exclusão de registros  
- Integração com API  
- Estrutura preparada para expansão (usuários, autenticação, etc.)  

---

## 🎯 Objetivo

Projeto desenvolvido para prática e consolidação de conhecimentos em desenvolvimento fullstack, com foco em criação de APIs, consumo de dados e organização de código.
